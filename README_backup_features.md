# 🤖 AliExpress Bot

A powerful Telegram bot for managing and publishing AliExpress products with affiliate links.

## Features

✅ **Product Management**
- Add and manage AliExpress products
- Store product details (name, link, price, description)
- Edit and delete products

✅ **Automatic Publishing**
- Publish products to your Telegram channel
- Automatic affiliate link insertion
- Professional marketing messages in English

✅ **Discount Management**
- Create and manage discount codes
- Percentage-based discounts
- Track active promotions

✅ **Channel Monitoring**
- Monitor saved posts for new products
- Auto-publish from drafts
- Scheduled publishing

✅ **Statistics & Analytics**
- Track published products
- View channel performance
- Monitor engagement metrics
